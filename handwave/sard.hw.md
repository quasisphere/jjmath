# Local Sard Theorem

This note isolates the Euclidean Sard input needed to choose smooth boundary
levels in the annular-exhaustion construction.  The required result is only
local and two-dimensional: for a smooth real-valued function on an open subset
of the complex plane, the critical values attained on a bounded subset have
one-dimensional Lebesgue measure zero.
